## NANO BANANA PRO EXECUTION STEPS

### Step 1: Generate Base Images
For each of the 16 prompts:
1. Open Nano Banana Pro
2. Copy prompt from .md file
3. Set resolution: 3840x2160 (4K)
4. Set style: Photorealistic
5. Generate 2-3 variations per prompt
6. Save to: `output/images/[filename].jpg`

### Step 2: Create Aspect Ratio Variations
For each base image, create:
- **16:9** (3840x2160) - Facebook/LinkedIn
- **4:5** (1080x1350) - Instagram Feed
- **9:16** (1080x1920) - Instagram Stories/Reels

### Step 3: Add Text Overlays
Use Canva, Photoshop, or Figma to add:
- Headline (bold, 60-72pt)
- Subheadline (regular, 32-36pt)
- CTA Button (AT&T blue #00A8E0)
- Disclaimer (small, 10-14pt)

### Step 4: Brand Compliance Review
Check each image for:
- [ ] AT&T blue color (#00A8E0) present
- [ ] Professional quality maintained
- [ ] Text readable on image
- [ ] Disclaimer legible
- [ ] Price/offer accurate
